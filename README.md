
# Course Recovery System (CRS) for Educational Institutions

## Overview
The Student Course Recovery System (CRS) is a Java-based software solution designed to help students at institutional educations recover their incomplete or failed courses, ensuring their timely graduation. 

The platform serves as a centralized platform for administrative staff to monitor student performance and deployed structured recovery plans as well as audit students for their eligibility to move on to the next level of study.
## Features

- Role-Based Access Control
- CRUD Operations for User Management and Student Recovery Plans
- Eligibility Auditing
- PDF Reporting 
- Automated Email Notifications

## Techniques 

- Object-Oriented Programming
- Modular Programming
- Java Swing for GUI
- SMTP Integration
- Document Generation
- Password Hashing & Audit Trails (Auth)
